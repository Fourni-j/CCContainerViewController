# CCContainerViewController
Custom UITabBarViewController for iPad

## Installation

Use [CocoaPods](http://cocoapods.org)

```sh
pod 'CCContainerViewController', :git => 'https://github.com/Fourni-j/CCContainerViewController.git'
```
